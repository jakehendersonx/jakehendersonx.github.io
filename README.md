
## Installation
install ruby
```
brew install ruby
```
now you have gem, get bundler
```
gem install bundler
```
or update if need be
```
gem update bundler
```
install from `Gemfile`
```
bundle install
```
check version
```
bundle exec jekyll -v
```
run site locally
```
bundle exec jekyll serve
```
